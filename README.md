# Hello, Dear Reader
##### This is a manual instructions to use my "SIMPLE" Calculator in Rust

1. Open Your Terminal
2. install git if you don't have it

Clone This Repository :
```
git clone https://github.com/saif-alden2error/Simple-Qualcalator.git
```

Enter Inside The Directory :
```
cd Simple-Qualcalator
```


Run The Program :
```
cargo run -q
```

To EXIT the program JUST hit CTRL + C or any other character makes this program errors
